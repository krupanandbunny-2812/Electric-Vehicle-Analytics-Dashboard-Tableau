1. Project Title / Headline

2. Short Description / Purpose
The Electric Vehicle Analytics Dashboard is a dynamic and interactive Tableau visualization that explores EV adoption patterns, performance characteristics, and geographic distribution.
It helps analysts, policymakers, and business stakeholders understand how electric vehicles are evolving across regions and categories to support strategic decision-making in the EV ecosystem.

4. Data Source
Source: Public Electric Vehicle datasets (government & open-source platforms)
The dataset includes electric vehicle records covering:

Vehicle categories and types

Adoption counts and trends

Performance-related attributes

Geographic information (state/region-level data)

The data is structured to support category-wise, time-based, and regional analysis of EV growth and performance.
